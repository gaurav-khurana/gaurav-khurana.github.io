The Game Vault

Web application for online purchase and rental of video games for Playstation & Xbox One console.
Responsive across all devices.
Client side built using React, Nextjs, Axios & SASS.
Server side built using Node, Express, Knex and MYSQL database.
Deployment using Netlify and Heroku.
